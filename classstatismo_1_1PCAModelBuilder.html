Detailed Description

template<typename T>
class statismo::PCAModelBuilder< T >

Create StatisticalModel using Principal Component Analysis.

This class implements the classical PCA based approach to Statistical Models.

Build a new model from the training data provided in the dataManager.

Parameters
sampleDataListsample set holding the data
noiseVariancevariance of N(0, noiseVariance) distributed noise on the points If this parameter is set to 0, we have a standard PCA model. For values > 0 we have a PPCA model.
computeScoresdetermines whether the scores (the pca coefficients of the examples) are computed and stored as model info (computing the scores may take a long time for large models).
methodSpecifies the method which is used for the decomposition
