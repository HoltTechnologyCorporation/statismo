unsigned  componentInd 
) const
inlineoverridevirtual

Define the mapping between the point ids and the position in the vector.

Assuming for example that a 3D mesh type is represented. A conversion strategy used in DatasetToSampleVector could be to return a vector . In this case, this method would return for inputs ptId, componentId the value ptId * 3 + componentId
