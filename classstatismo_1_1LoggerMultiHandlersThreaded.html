Detailed Description

Logger class with background threading capabilities and able to handle multiple log handlers.

This class is responsible for all the log mechanism except writing and formatting dedicated to handlers register in the logger.

The logger dispatches a log to specific handlers according to the log level. To know if a log should be passed or not, the logger checks:

  • its base level of filtering
  • the specific level of filtering bound to a given log handler

Logging can be done in background if parallelism is set.
